------------------------------------------------------------------------------
 DiskMark.exe    : NT - x86 - Unicode
 DiskMarkX64.exe : NT - x64 - Unicode

------------------------------------------------------------------------------
 重要事項 
------------------------------------------------------------------------------
 "MB/s" は 1,000,000byte/s の意味で使用しています。
 1024*1024byte/s ではないのでご注意ください。
